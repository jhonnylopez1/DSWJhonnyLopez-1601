FlexBox

Flexbox es una tecnologia de diseño web que permite crear diseños flexibles y responsivos en paginas web. Con Flexbox, los elementos dentro de un contenedor pueden ser organizados y distribuidos de manera dinamica, segun el espacio disponible y las necesidades del diseño. Flexbox utiliza un sistema de cajas flexibles, donde cada elemento dentro del contenedor puede ser manipulado de forma individual o como parte de un gupo. Esto prmite controlar la alineacion, distribucion y tamaño de los elementos, asi como la direccionen la que se distribuyen(horizontal o vertical)
